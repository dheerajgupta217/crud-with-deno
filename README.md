# crud-with-deno
The complete getting started tutorial can be found at:


### Install Deno

```bash
curl -fsSL https://deno.land/x/install/install.sh | sh
```

### Run the project

```bash
deno run --allow-net server.ts
```

## Controller Routes

```
GET            /employee
GET BY ID      /employee/<id>
POST           /employee
UPDATE BY ID   /employee/<id>
DELETE BY ID   /employee/<id>
```
